# BX3 React Example App

Demonstrates the BX3 Design System React integration.

## Quick Start

```bash
# From the react-example directory
npm install
npm run dev
```

Open http://localhost:5173

## Features Demonstrated

- **Theme Switching** — Toggle between Zospace, AgentOS, VPC, and Irrig8 themes
- **Button Variants** — Primary, Secondary, Ghost, Danger
- **Button Sizes** — Small, Medium, Large
- **Button States** — Loading, Disabled, Full Width
- **Accessibility** — Keyboard navigation, ARIA labels, screen reader support

## Code Structure

```
src/
├── App.tsx          # Main demo application
├── main.tsx         # Vite entry point
└── index.html       # HTML template
```

## Building for Production

```bash
npm run build
npm run preview
```
